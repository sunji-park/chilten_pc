## 1. 서비스 개요

### 칠텐(ChilTen)이란?

볼링 기록, 클럽, 커뮤니티 서비스를 아우르는 **대한민국 최대 볼링 전문 플랫폼**

| 지표 | 수치 |
|------|------|
| 회원수 | **12만명** |
| 볼링 클럽수 | **6,000개** |
| 등록 볼링장수 | **전국 1,024개** |
| 커뮤니티 게시글 | **50만건+** |

### 앱 주요 기능
- **마이볼링**: 개인 볼링 기록 (에버, 올커버, 200UP, 베스트스코어)
- **커뮤니티**: 중고장터, 자유게시판, 이벤트, 클럽홍보, 대회소식
- **랭킹**: 지역별/성별 기록 랭킹, 클럽 에버 랭킹, 명예의 전당
- **볼링장**: 전국 볼링장 정보 (매우 정확하게 정리됨)
- **미션/핀**: 출석/획득미션 포인트, 볼뽑기 드로우 이벤트

### 웹 버전 런칭 배경

| 구분 | 내용 |
|------|------|
| 목적 | 모바일 앱의 커뮤니티 활성화를 웹 환경으로 확장 |
| 기회 | 중고장터, 대회 정보 등 PC에서 더 편리한 기능 존재 |
| BM | 볼링장 홈페이지 구독 서비스로 수익화 |
| 1차 목표 | 커뮤니티 + 볼링장 정보 웹 최적화 |

## 2. 1차 개발 범위

### Phase 1 포함 항목

```
칠텐 웹 v1
├── 커뮤니티
│   ├── 중고장터 (지역 기반 소팅)
│   ├── 자유게시판

│   └── 대회소식게시판
└── 볼링장
    ├── 지도뷰 (Kakao Maps 연동, 전국/지역별)
    ├── 목록뷰 (지역/필터별)
    ├── 상세페이지 (상세 정보 항목)

```

- 마이볼링 (기록 입력/조회)
- 랭킹 시스템

- 채팅 기능
- 인앱결제/핀 구매

## 5. 커뮤니티 기능 정의

### 5.1 중고장터 ⭐ 핵심 기능

**특징**: 앱에서 가장 활발한 게시판. PC에서 편의성 대폭 향상.

**지역 기반 소팅 시스템**
```
사용자 지역 설정 (내 지역: 서울 강남구)
     ↓
정렬 우선순위:
  1순위: 동일 지역구 (서울 강남구)  →  "내 지역" 초록 뱃지
  2순위: 동일 시/도 (서울)
  3순위: 나머지 (등록일순)
```

**주요 기능**

| 기능 | 설명 |
|------|------|
| 카테고리 | 볼 / 볼링화 / 볼링가방 / 장갑·용품 / 기타 |
| 거래상태 | 판매중 / 예약중 / 판매완료 |
| 정렬 | 내 지역 근처 우선 / 최신순 / 가격낮은순 / 가격높은순 |
| 상품 정보 | 사진, 제목, 가격, 지역, 거래상태, 관심수 |
| 상세페이지 | 상세사진, 설명, 판매자 프로필, 채팅문의 |

**상품 목록 레이아웃**: 4열 카드 그리드 (와이드 화면 최적화)

### 5.2 자유게시판

| 기능 | 설명 |
|------|------|
| 목록 | 번호 / 제목 / 작성자 / 조회수 / 댓글수 / 날짜 |
| 정렬 | 최신순 / 조회수순 / 댓글순 |
| 검색 | 제목+내용 검색 |
| 작성 | 텍스트+이미지 첨부, 글쓰기 버튼 |

### 5.3 이벤트게시판

볼링장에서 진행 중인 이벤트 정보 모음

| 기능 | 설명 |
|------|------|
| 필터 | 지역별 (전체/서울/경기/인천/부산/대구/대전/광주) |
| 날짜 | 월별 필터 |
| 이벤트 종류 | 할인 / 무료 / 신규회원 / 기타 |
| 카드 정보 | 볼링장명, 이벤트명, 기간, 진행상태 뱃지 |
| 작성 권한 | 볼링장 어드민 계정만 등록 가능 |

### 5.4 클럽홍보게시판

볼링 클럽의 신규 회원 모집 전용 게시판

| 기능 | 설명 |
|------|------|

### 5.5 대회소식게시판

아마추어 볼링 대회 정보 모음

| 기능 | 설명 |
|------|------|
| 필터 | 지역별 / 대회종류(오픈·클럽·협회) |
| 테이블 | 대회명, 주최, 날짜, 장소, 참가비, D-day |
| 신청 | 대회별 신청하기 버튼 (외부링크 또는 내부) |

## 6. 볼링장 기능 정의

### 6.1 지도뷰

**지도 API**: Kakao Maps JavaScript SDK (프로토타입: Leaflet.js + OpenStreetMap)

| 기능 | 설명 |
|------|------|
| 기본뷰 | 전국 지도, 줌 레벨 7 (한반도 전체 보임) |
| 마커 | 칠텐제휴(파란 마커) / 일반(초록 마커) |
| 클러스터링 | 줌아웃 시 인근 볼링장 군집화 |
| 마커 팝업 | 볼링장명, 주소, 레인수, 요금, [상세보기] |
| 좌측 패널 | 지역 필터 칩 + 볼링장 리스트 |
| 검색 | 볼링장명 검색 |
| 현재위치 | GPS 기반 내 주변 볼링장 |

**지역 필터**: 전체 / 서울 / 경기 / 인천 / 부산 / 대구 / 대전 / 광주 / 강원 / 충청 / 전라 / 경상 / 제주

### 6.2 목록뷰

| 기능 | 설명 |
|------|------|
| 지역 선택 | 시/도 → 시/군/구 2단계 선택 |
| 정렬 | 거리순 / 이름순 / 레인수순 |
| 필터 칩 | 칠텐제휴 / 24시간 / 주차가능 / 샤워실 / 프로샵 |
| 카드 정보 | 썸네일, 볼링장명, 주소, 영업상태, 레인수, 핀세터, 레인타입, 요금 |
| 액션 | 전화하기 / 상세보기 |

### 6.3 상세페이지

볼링장별 상세 정보 항목:

**헤더 배지**
- `칠텐제휴` (초록) — 칠텐 제휴 볼링장
- `영업중 / 영업종료` — 실시간 영업 상태
- `⚡ 칠텐자동기록` (보라, #6D28D9) — 칠텐 자동기록 시스템 탑재 볼링장에 추가 표시

**상단 사진 스와이퍼**
- 최대 5장 등록 (어드민에서 관리)
- 좌/우 화살표 + 하단 dot indicator + 우상단 N/5 카운터
- 가로 슬라이드, CSS transition 기반
- 모바일 최적화 (높이 58vw)

**기본 정보**
- 볼링장명, 주소, 전화번호
- 영업시간 (요일별 오픈/마감)
- 이용요금: 텍스트 표기 + **🖼️ 이미지로 보기** 버튼 (클릭 시 모달로 이미지 표시)
- 대화료

**시설 정보**
- 레인수
- 레인타입 (합성 / 우레탄)
- 핀세터 유형 (AMF / Brunswick / QubicaAMF)
- 시설타입 (일반 / 복합)
- 편의시설 (주차 / 샤워실 / 프로샵 / 카페 / 락커)

**칠텐 연동 정보**
- 칠텐 방문 회원수
- 총 기록 게임수
- 평균 에버
- 200UP 비율

**부가 정보**
- 상주 클럽 목록
- 현재 이벤트
- 위치 지도
- 평점 (칠텐 회원 리뷰 기반)

**우측 사이드바**
- 🌐 **홈페이지 바로가기** — 볼링장 전용 홈페이지를 새 탭으로 오픈 (기존 전화하기 버튼 대체)
  > 볼링장 홈페이지는 프리미엄 플랜에서 제공하는 전용 홍보 페이지
- 위치 지도 (Kakao Maps 연동 예정)
- 칠텐 평점 (별점 분포)
- 앱 다운로드 유도 배너

### 6.4 볼링장 어드민

볼링장별 전용 관리자 페이지 (어드민 계정 기반)

**탭 구성**

| 탭 | 기능 |
|----|------|
| 정보관리 | 기본정보, 영업시간, 요금, 시설정보 편집 / **이용요금 이미지 등록** (이미지로 보기 모달에 표시) |
| 이벤트관리 | 이벤트 등록/수정/삭제 |
| 사진관리 | 볼링장 사진 업로드/삭제/정렬 (**최대 5장**, 상세페이지 스와이퍼에 노출) |
| 통계 | 방문 칠텐 유저수, 기록수, 팔로워, 이벤트 조회수 |
| 구독플랜 | 현재 구독 플랜 확인 및 업그레이드 |

## 10. 기술 스택

### Frontend
```
- Framework: React 18 + TypeScript
- State: Zustand (가벼운 상태관리)
- Router: React Router v6
- Map: Kakao Maps JavaScript SDK
- Styling: Tailwind CSS + CSS Variables
- HTTP: Axios
- Build: Vite
```

### Backend
```
- Runtime: Node.js + Express (또는 NestJS)
- Database: PostgreSQL (메인) + Redis (캐시/세션)
- ORM: Prisma
- Auth: JWT + Refresh Token
- File: AWS S3 (이미지 업로드)
```

### DevOps
```
- Hosting: AWS (EC2 + RDS + S3 + CloudFront)
- CI/CD: GitHub Actions
- Container: Docker + Docker Compose
- Monitoring: Sentry + CloudWatch
```

### 외부 API
```
- Kakao Maps JavaScript API (지도)
- Kakao 소셜 로그인
- NHN KCP 또는 토스페이먼츠 (결제)
- Firebase (푸시 알림)
```
