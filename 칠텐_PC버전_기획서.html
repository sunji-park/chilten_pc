서비스 개요

칠텐 소개

칠텐은 대한민국 최대 규모의 보링 커뮤니티 플랫폼으로, 활발한 사용자 기반과 풍부한 정보를 갖추고 있습니다.

칠텐 플랫폼 현황
  • 누적 회원 수: 12만명
  • 등록 클럽 수: 6천개
  • 보유 볼링장 정보: 전국 1,000여개
  • 월간 활성 사용자: 5만명 이상
  • 현재 서비스: 모바일 앱 중심

현재 서비스 특징

  • 커뮤니티: 자유로운 정보 공유와 상호작용
  • 데이터베이스: 전국 볼링장 정보 검색 및 조회
  • 클럽 네트워크: 보링 애호가들의 클럽 커뮤니티
  • 기록 관리: 개인 볼링 기록 통계 및 추적
  • 중고 거래: 보링 용품 B2C 마켓플레이스

커뮤니티 기능 정의

칠텐의 커뮤니티는 5개의 특성화된 게시판으로 구성되어 있습니다. 각 게시판은 독립적인 기능을 제공하면서도 통합된 사용자 경험을 제공합니다.

4-1. 중고장터

보링 용품을 사고팔 수 있는 B2C 마켓플레이스입니다. 안전한 거래를 위해 체계적인 상태 관리와 채팅 기능을 제공합니다.

주요 기능

기능 상세 설명 화면 요소
카테고리 볼, 신발, 가방, 기타 용품으로 분류 탭/드롭다운 필터
글쓰기 상품명, 가격, 설명, 사진 최대 5장 첨부 폼 기반 입력 화면
검색 상품명, 설명으로 검색 검색바, 자동완성
거래 상태 판매중 / 예약중 / 판매완료 상태 배지, 상태 전환 버튼
가격 표시 정가 대비 할인율 표시, 가격 범위 필터 가격 태그, 슬라이더 필터
채팅 문의 판매자와 실시간 채팅 채팅 아이콘, 채팅창
거래 평가 거래 완료 후 평가 (5성) 별점, 후기 작성

데이터 모델

항목 데이터 필수 여부
상품명 텍스트 필수
카테고리 선택 (볼/신발/가방/기타) 필수
가격 숫자 (원) 필수
상품 설명 장문 텍스트 필수
사진 이미지 (최대 5장) 필수
거래 상태 판매중/예약중/판매완료 자동 설정
작성자 사용자 정보 자동 설정
작성일시 타임스탬프 자동 설정

4-2. 자유게시판

보링 일상을 공유하고 사람들과 소통하는 공간입니다. 자유로운 주제의 글을 작성하고 댓글로 상호작용합니다.

주요 기능

기능 상세 설명 화면 요소
글쓰기 제목, 본문, 태그 입력 에디터 (텍스트 기반)
댓글 글 작성자와 댓글 작성자 간의 상호작용 댓글 입력칸, 댓글 목록
좋아요 글과 댓글에 좋아요 기능 하트 아이콘, 좋아요 수
검색 제목, 본문, 작성자, 태그로 검색 검색바, 검색 결과
정렬 최신순, 인기순, 댓글순 정렬 드롭다운

게시글 표시 형식

  • 리스트 뷰: 제목, 작성자, 작성일, 댓글수, 좋아요수
  • 상세 뷰: 전체 본문, 댓글, 추천 게시글

4-3. 이벤트게시판

전국 보링장에서 진행하는 다양한 이벤트 정보를 한 곳에서 확인할 수 있습니다. 캘린더 뷰로 일정을 관리할 수 있습니다.

주요 기능

기능 상세 설명
이벤트 정보 보링장명, 이벤트 기간, 이벤트 내용/설명
지역 필터 시/도별로 이벤트 필터링
캘린더 뷰 월간/주간 캘린더에 이벤트 표시
글쓰기 보링장 사업자가 이벤트 등록
좋아요/스크랩 관심 이벤트 저장

4-4. 클럽홍보게시판

보링 클럽들이 자신의 클럽을 소개하고 회원을 모집하는 공간입니다. 클럽 검색 및 가입 문의 기능을 제공합니다.

주요 기능

기능 상세 설명
클럽 정보 클럽명, 회원수, 에버/수준(초급/중급/상급), 활동지역, 소개
필터 지역, 수준, 성별(남성/여성/혼성), 활동빈도
글쓰기 클럽 소개글 작성, 이미지 첨부
가입 문의 클럽 관리자에게 가입 문의 메시지 전송
클럽 정보 조회 회원수, 에버, 활동지역 등 상세 정보

4-5. 대회소식게시판

아마추어 보링 대회 정보를 공유하는 게시판입니다. 대회 참가를 위한 모든 정보를 제공합니다.

주요 기능

기능 상세 설명
대회 정보 주최사, 대회명, 개최 날짜/기간, 개최 장소(보링장), 참가비, 참가 방법
대회 종류 오픈 대회 / 클럽 대회 / 협회 대회
지역 필터 시/도별로 대회 필터링
글쓰기 대회 주최자가 대회 정보 등록
참가 신청 대회 참가 신청 (별도의 연락 방법 표시)
5

볼링장 기능 정의

칠텐의 핵심 기능인 볼링장 정보 조회 시스템입니다. 사용자 친화적인 인터페이스로 원하는 볼링장을 빠르게 찾을 수 있습니다.

5-1. 지도 뷰

전국의 모든 볼링장을 지도상에 시각화하여 위치 기반의 직관적인 검색을 제공합니다.

주요 기능

기능 상세 설명
기본 맵 대한민국 전국 지도, 줌/팬 가능
마커 클러스터 확대 수준에 따라 마커 그룹핑
현재 위치 사용자의 GPS 위치 표시 및 주변 볼링장 자동 표시
마커 팝업 마커 클릭 시 팝업: 보링장명, 주소, 레인수, 영업 여부
상세 페이지 이동 팝업 또는 목록에서 상세 페이지로 이동

UI/UX 설계

  • 지도는 화면의 70% 이상 차지
  • 우측 사이드 패널: 검색 필터 및 결과 리스트
  • 마커 아이콘: 영업중/폐업 상태에 따른 시각적 구분
  • 현재 위치: 파란색 원으로 표시

5-2. 리스트 뷰

필터링과 정렬을 통해 원하는 조건의 볼링장을 체계적으로 검색할 수 있습니다.

주요 기능

기능 상세 설명 기술
지역 필터 시/도 선택 > 시/군/구 선택 (계단식 필터) 드롭다운
정렬 거리순 / 이름순 / 레인수순 라디오 버튼
검색 보링장명으로 검색 검색바, 자동완성
카드형 리스트 각 보링장 카드: 사진, 이름, 주소, 레인수, 평점 그리드 또는 리스트
제휴 뱃지 칠텐 제휴 보링장 표시 배지 아이콘

카드 요소

  • 대표 이미지
  • 보링장명 + 제휴 뱃지
  • 주소 (지역)
  • 레인수 (예: "30레인")
  • 평점 (사용자 평가)
  • 클릭 시 상세 페이지로 이동

5-3. 상세 페이지
