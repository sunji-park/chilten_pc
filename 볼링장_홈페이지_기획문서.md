## 1. 서비스 개요

### 1.1 프로젝트 목적

칠텐 PC 홈페이지의 **볼링장 상세페이지 > "홈페이지 바로가기"** 버튼 클릭 시 **새 창(새 탭)**으로 열리는 볼링장 전용 홍보 사이트를 구축한다.

### 1.2 핵심 컨셉

볼링장 홍보에 최적화된 사이트로, 볼링장의 **사진을 적극 활용**하여 볼링장별 **차별점이 시각적으로 드러나도록** 설계한다. 진행 중인 이벤트를 돋보이게 노출하고, 일반 정보를 체계적으로 안내한다.

### 1.3 핵심 원칙

- 볼링장 사진이 메인 비주얼 역할을 하여 첫인상을 결정
- 이벤트 정보가 메인 페이지에서 즉시 눈에 띄도록 배치
- 각 볼링장이 자체 브랜드 컬러를 설정할 수 있어 개성 있는 홈페이지 운영 가능
- 칠텐 회원 DB를 공유하여 별도 가입 불필요
- 점주에게 별도 어드민(새 창)을 제공하여 자율적 관리

### 1.4 타겟 사용자

| 사용자 | 목적 |
|--------|------|
| 볼링장 방문 고객 | 볼링장 정보 확인, 이벤트 확인, 공지 확인 |
| 상주 클럽 회원 | 클럽 공지 확인, 클럽 일정 확인 |
| 볼링장 점주 (어드민) | 홈페이지 관리, 알림 발송, 회원 관리 |

## 2. 메뉴구조도 (IA)

### 2.1 볼링장 홈페이지 (방문자)

```
볼링장 홈페이지
├── 홈 (메인)
│   ├── 히어로 영역 (사진 슬라이더)
│   ├── 진행 중 이벤트 하이라이트
│   ├── 볼링장 한줄 소개
│   └── 빠른 메뉴 (영업시간/요금/위치)
├── 소개
│   ├── 볼링장 소개
│   ├── 시설 안내 (사진 갤러리)
│   ├── 이용 요금표
│   └── 오시는 길 (지도)
├── 이벤트
│   ├── 진행 중 이벤트
│   └── 종료된 이벤트
├── 공지사항
│   └── 공지 게시판 (목록 / 상세)
├── 상주클럽
│   ├── 클럽 목록
│   └── 클럽 상세 (회원수, 활동정보)
└── 갤러리
    └── 볼링장 사진/영상 모음
```

### 2.2 볼링장 어드민 (점주 전용 — 별도 새 창)

```
볼링장 어드민
├── 대시보드
│   ├── 방문자 통계
│   ├── 알림 발송 현황
│   └── 최근 활동 요약
├── 일반 정보 관리
│   ├── 기본 정보 (이름, 주소, 전화, 영업시간)
│   ├── 이용 요금 관리
│   ├── 시설 정보 (레인수, 편의시설 등)
│   └── 소개글 편집
├── 사진/갤러리 관리
│   ├── 메인 사진 관리 (히어로 슬라이더)
│   ├── 시설 사진 관리
│   └── 갤러리 관리
├── 게시판 관리
│   ├── 공지사항 관리 (작성/수정/삭제)
│   └── 이벤트 관리 (등록/수정/종료)
├── 상주클럽 관리
│   ├── 클럽 가입 승인/거절
│   └── 등록된 클럽 목록
├── 알림 발송
│   ├── 상주클럽원 알림
│   ├── 방문/단골 고객 알림
│   ├── 발송 이력 조회
│   └── 발송 채널 설정 (푸시/카카오/SMS)
└── 관리자 계정 관리
    ├── 관리자 목록
    ├── 관리자 추가
    └── 관리자 삭제
```

## 3. 페이지별 상세 기획

### 3.1 홈 (메인 페이지)

**역할**: 볼링장의 첫인상을 결정하는 핵심 페이지. 사진과 이벤트가 주역.

**레이아웃 구성**

| 영역 | 설명 | 높이(vh) |
|------|------|----------|
| **헤더** | 볼링장 로고/이름 + 네비게이션 | 고정 |
| **히어로 슬라이더** | 볼링장 대표 사진 풀와이드 슬라이드 (최대 5장, 자동 재생) | 60vh |
| **이벤트 하이라이트** | 진행 중 이벤트 카드 (최대 3개, 가로 스크롤) | 자동 |
| **볼링장 소개** | 한줄 소개 + 핵심 정보 카드 (레인수, 편의시설 아이콘) | 자동 |
| **영업 안내** | 영업시간, 요금 요약, 위치 미니맵 3열 그리드 | 자동 |
| **갤러리 프리뷰** | 최근 사진 6장 그리드 + "더보기" | 자동 |
| **푸터** | 볼링장 연락처, 주소, 칠텐 로고, SNS | 고정 |

**히어로 슬라이더 사양**

- 풀와이드 (100vw) 이미지
- 자동 재생 (5초 간격)
- 좌/우 화살표 네비게이션
- 하단 dot indicator
- 오버레이 텍스트: 볼링장명 + 슬로건 (반투명 배경)
- 모바일: 높이 40vh로 조정

**이벤트 하이라이트 사양**

- 카드 형태 (이미지 + 제목 + 기간 + "자세히 보기" 버튼)
- 이벤트가 없을 경우: "현재 진행 중인 이벤트가 없습니다" 표시
- "진행중" 뱃지 (빨간색 펄스 애니메이션)
- D-day 카운터 표시

### 3.2 소개 페이지

**역할**: 볼링장의 상세 정보를 체계적으로 안내

**섹션 구성**

1. **볼링장 소개**: 점주가 작성한 소개글 (WYSIWYG 에디터 기반)
2. **시설 안내**: 사진 갤러리 (라이트박스) + 시설 아이콘 그리드
   - 아이콘: 레인수, 레인타입, 핀세터, 주차, 샤워실, 프로샵, 카페, 락커
3. **이용 요금**: 테이블 형태 + "요금표 이미지 보기" 버튼 (모달)
   - 게임 요금, 대화료 등 항목별 정리
4. **오시는 길**: Kakao Maps 임베드 + 주소 + 대중교통 안내

### 3.3 이벤트 페이지

**역할**: 볼링장 이벤트를 돋보이게 노출

**목록 레이아웃**

- 탭: 진행 중 / 종료됨
- 카드 그리드 (2열)
- 카드 구성: 이벤트 이미지(16:9), 제목, 기간, 상태 뱃지, 요약 텍스트
- 진행 중 이벤트: 상단 고정 + "HOT" 뱃지
- 종료된 이벤트: 흑백 처리 + "종료" 뱃지

**상세 페이지**

- 이벤트 이미지 (풀와이드)
- 제목, 기간, 상세 설명 (HTML)
- 참여 방법/조건
- 관련 문의 연락처

### 3.4 공지사항 페이지

**역할**: 볼링장 공지사항 게시판

**목록**

| 컬럼 | 설명 |
|------|------|
| 번호 | 자동 증가 |
| 제목 | 클릭 시 상세 이동 |
| 작성일 | YYYY.MM.DD |
| 조회수 | 클릭 카운트 |

- 고정 공지: 상단 고정 (핀 아이콘)
- 페이지네이션: 15건/페이지
- 검색: 제목 기반

**상세**

- 제목, 작성일, 조회수
- 본문 (텍스트 + 이미지)
- 이전글/다음글 네비게이션
- 목록으로 돌아가기 버튼

### 3.5 상주클럽 페이지

**역할**: 해당 볼링장에 상주하는 볼링 클럽 소개

**목록**

- 카드 그리드 (3열)
- 카드 구성: 클럽 로고/대표사진, 클럽명, 회원수, 평균에버, 활동요일
- "가입 문의" 버튼

**클럽 상세**

- 클럽 소개글
- 활동 일정 (요일/시간)
- 회원 구성 (총원, 평균에버, 성별 비율)
- 가입 조건
- 연락처/문의 방법

### 3.6 갤러리 페이지

**역할**: 볼링장 사진 및 영상 모음

- 사진 그리드 (4열, 반응형)
- 라이트박스 뷰 (좌우 네비게이션)
- 카테고리 필터: 전체 / 시설 / 이벤트 / 클럽활동

## 4. 템플릿 시스템

### 4.1 개요

5개의 템플릿을 제공하며, 각 볼링장이 원하는 템플릿을 선택하고 자체 브랜드 컬러를 설정할 수 있다.

### 4.2 공통 커스터마이징 옵션

| 항목 | 설명 |
|------|------|
| **메인 컬러** | 볼링장 브랜드 컬러 (헤더, 버튼, 강조색) |
| **서브 컬러** | 배경, 호버 등 보조색 |
| **로고** | 볼링장 자체 로고 업로드 |
| **대표 사진** | 히어로 슬라이더 사진 (최대 5장) |
| **소개글** | 볼링장 자유 소개 텍스트 |
| **슬로건** | 한줄 슬로건 |

### 4.3 템플릿 정의

#### 템플릿 1: 모던 심플 (Modern Simple)

- **컨셉**: 깔끔하고 미니멀한 정보 중심 레이아웃
- **특징**: 넓은 여백, 깔끔한 타이포그래피, 카드 기반 UI
- **히어로**: 단일 이미지 + 중앙 정렬 텍스트 오버레이
- **컬러**: 화이트 베이스 + 포인트 컬러 1개
- **적합**: 고급 이미지의 프리미엄 볼링장

#### 템플릿 2: 다이나믹 스포츠 (Dynamic Sports)

- **컨셉**: 역동적이고 에너지 넘치는 스포츠 감성
- **특징**: 대각선 분할 레이아웃, 볼드 타이포, 애니메이션
- **히어로**: 풀스크린 슬라이더 + 동적 텍스트 효과
- **컬러**: 다크 베이스 + 비비드 액센트 컬러
- **적합**: 젊은 층 타겟, 활기찬 분위기의 볼링장

#### 템플릿 3: 클래식 엘레강스 (Classic Elegance)

- **컨셉**: 전통적이고 격조 있는 클래식 디자인
- **특징**: 세리프 폰트, 금/어두운 톤, 테두리 장식
- **히어로**: 사진 + 하단 정보 바
- **컬러**: 다크네이비/블랙 + 골드/실버 액센트
- **적합**: 전통 있는 볼링장, 고급 이미지

#### 템플릿 4: 프레시 팝 (Fresh Pop)

- **컨셉**: 밝고 친근한 팝 컬러 디자인
- **특징**: 둥근 모서리, 일러스트 아이콘, 밝은 배경
- **히어로**: 사진 + 말풍선 스타일 텍스트
- **컬러**: 파스텔 베이스 + 팝 컬러 포인트
- **적합**: 가족 친화적, 초보자 환영 볼링장

#### 템플릿 5: 네온 나이트 (Neon Night)

- **컨셉**: 볼링장 특유의 네온/형광 감성
- **특징**: 다크 배경 + 네온 글로우 효과, 그라디언트
- **히어로**: 다크 오버레이 + 네온 타이포그래피
- **컬러**: 블랙/다크그레이 + 네온 핑크/그린/블루
- **적합**: 야간 볼링, 엔터테인먼트 볼링장

## 5. 어드민 기능 정의

### 5.1 접근 방식

- 별도 URL로 새 창에서 열림: `{볼링장도메인}/admin`
- 칠텐 회원 계정으로 로그인 후, 해당 볼링장의 관리자 권한 확인
- 어드민 권한이 없는 계정은 접근 불가 (403)

### 5.2 대시보드

| 지표 | 설명 |
|------|------|
| 오늘 방문자 | 홈페이지 일일 방문자 수 |
| 이번 주 방문자 | 주간 방문자 추이 그래프 |
| 활성 이벤트 | 현재 진행 중인 이벤트 수 |
| 미처리 알림 | 승인 대기 중인 클럽 가입 요청 수 |
| 최근 공지 | 최근 등록된 공지사항 |
| 알림 발송 현황 | 최근 발송한 알림 요약 |

### 5.3 게시판 관리

**공지사항 관리**

| 기능 | 설명 |
|------|------|
| 목록 조회 | 전체 공지사항 목록 (날짜, 제목, 조회수, 상태) |
| 공지 작성 | 제목, 본문(WYSIWYG), 이미지 첨부, 고정여부 |
| 공지 수정 | 기존 공지 내용 편집 |
| 공지 삭제 | 삭제 확인 후 제거 |

**이벤트 관리**

| 기능 | 설명 |
|------|------|
| 이벤트 등록 | 이미지, 제목, 기간, 상세내용, 참여방법 |
| 이벤트 수정 | 진행 중/종료 상태 변경, 내용 수정 |
| 이벤트 삭제 | 삭제 확인 후 제거 |
| 이벤트 복제 | 기존 이벤트를 복제하여 새 이벤트 생성 |

### 5.4 알림 발송

**발송 대상 그룹**

| 대상 | 정의 |
|------|------|
| 상주클럽원 | 해당 볼링장에 등록된 클럽의 전체 회원 |
| 방문 고객 | 칠텐 앱에서 해당 볼링장을 방문(기록)한 고객 |
| 단골 고객 | 최근 3개월 내 3회 이상 방문한 고객 |
| 전체 | 위 모든 대상 |

**발송 채널**

| 채널 | 설명 | 비용 |
|------|------|------|
| 칠텐 앱 푸시 | 칠텐 앱 내 푸시 알림 발송 | 무료 |
| 카카오 알림톡 | 카카오톡 알림톡 API 연동 (템플릿 기반) | 건당 과금 |
| SMS | SMS 발송 서비스 연동 | 건당 과금 |

**발송 프로세스**

```
대상 그룹 선택 → 채널 선택 (복수 가능) → 메시지 작성 → 미리보기 → 발송 확인 → 발송
```

**발송 이력**

- 발송일시, 대상그룹, 채널, 메시지 요약, 발송건수, 성공/실패 건수

### 5.5 일반 정보 관리

| 항목 | 설명 |
|------|------|
| 기본 정보 | 볼링장명, 주소, 전화번호, 소개글, 슬로건 |
| 영업시간 | 요일별 오픈/마감 시간 설정 |
| 이용 요금 | 텍스트 요금 편집 + 요금표 이미지 업로드 |
| 시설 정보 | 레인수, 레인타입, 핀세터, 편의시설 체크 |
| 브랜드 설정 | 메인 컬러, 서브 컬러, 로고 업로드, 템플릿 선택 |

### 5.6 상주클럽 승인

| 기능 | 설명 |
|------|------|
| 가입 요청 목록 | 클럽명, 신청일, 회원수, 상태 |

| 거절 | 사유 입력 후 거절 |
| 클럽 해제 | 기존 상주 클럽 등록 해제 |

### 5.7 관리자 계정 관리

| 기능 | 설명 |
|------|------|
| 관리자 목록 | 현재 등록된 관리자 계정 목록 (이름, 이메일, 권한, 등록일) |
| 관리자 추가 | 칠텐 회원 검색 → 관리자 권한 부여 (소유자/매니저 역할) |

**권한 등급**

| 역할 | 권한 |
|------|------|
| 소유자 (Owner) | 모든 기능 + 관리자 추가/삭제 |
| 매니저 (Manager) | 게시판 관리, 알림 발송, 정보 관리 (관리자 계정 관리 제외) |

## 6. 알림발송 시스템

### 6.1 칠텐 앱 푸시 알림

- 칠텐 앱의 기존 Firebase Cloud Messaging (FCM) 인프라 활용
- 알림 클릭 시 앱 내 해당 볼링장 페이지로 이동
- 무료 (칠텐 인프라 비용)

### 6.2 카카오 알림톡

- 카카오 비즈니스 채널 연동
- 사전 등록된 템플릿 기반 발송
- 알림 템플릿 예시: 이벤트 안내, 공지 안내, 영업시간 변경 등
- 건당 과금 (약 8~15원)

### 6.3 SMS 발송

- NHN Cloud 또는 카카오 비즈메시지 SMS API 연동
- 단문(SMS) / 장문(LMS) 지원
- 건당 과금 (SMS 약 20원, LMS 약 50원)

### 6.4 알림 메시지 템플릿

| 유형 | 예시 |
|------|------|
| 이벤트 안내 | "[볼링장명] 새로운 이벤트! {이벤트명} ({기간})" |
| 공지 안내 | "[볼링장명] 공지사항: {제목}" |
| 영업시간 변경 | "[볼링장명] 영업시간이 변경되었습니다" |
| 자유 메시지 | 점주가 직접 작성하는 자유 형식 메시지 |

## 7. 회원 시스템 연동

### 7.1 칠텐 회원 DB 공유

- 별도 회원가입 없음
- 칠텐 앱/웹 계정으로 통합 로그인
- OAuth2 기반 SSO (Single Sign-On)

### 7.2 로그인 플로우

```
볼링장 홈페이지 → "로그인" 클릭
→ 칠텐 통합 로그인 페이지로 리다이렉트
→ 이메일/소셜 로그인 (카카오/네이버/구글)
→ 인증 완료 → 볼링장 홈페이지로 토큰과 함께 복귀
```

### 7.3 회원 정보 활용

|------|------|
| 닉네임, 프로필 | 공지사항 댓글 등에서 표시 |
| 지역 정보 | 알림 발송 대상 분류 |
| 방문 기록 | 방문 고객 / 단골 고객 판별 |
| 클럽 정보 | 상주 클럽원 판별 |

## 8. URL 및 접근 구조

### 8.1 URL 패턴

```
볼링장 홈페이지: https://bowling.chilten.com/{bowling-id}
볼링장 어드민:   https://bowling.chilten.com/{bowling-id}/admin
```

또는 서브도메인 방식:
```
볼링장 홈페이지: https://{bowling-id}.bowling.chilten.com
볼링장 어드민:   https://{bowling-id}.bowling.chilten.com/admin
```

### 8.2 접근 플로우

```
칠텐 PC 홈페이지
  └── 볼링장 상세페이지
       └── [🌐 홈페이지 바로가기] 버튼 클릭
            └── 새 탭으로 볼링장 전용 홈페이지 오픈

볼링장 어드민 접근:
  └── 칠텐 PC > 볼링장 어드민 메뉴
       └── [어드민 페이지 열기] 버튼 클릭
            └── 새 탭으로 볼링장 어드민 오픈
```

### 8.3 페이지 맵

```
/{bowling-id}                    → 메인 (홈)
/{bowling-id}/about              → 소개
/{bowling-id}/events             → 이벤트
/{bowling-id}/events/:eventId    → 이벤트 상세
/{bowling-id}/notices            → 공지사항
/{bowling-id}/notices/:noticeId  → 공지사항 상세
/{bowling-id}/clubs              → 상주클럽
/{bowling-id}/clubs/:clubId      → 클럽 상세
/{bowling-id}/gallery            → 갤러리
/{bowling-id}/admin              → 어드민 (새 창)
/{bowling-id}/admin/info         → 어드민 > 정보 관리
/{bowling-id}/admin/board        → 어드민 > 게시판 관리
/{bowling-id}/admin/notify       → 어드민 > 알림 발송
/{bowling-id}/admin/clubs        → 어드민 > 상주클럽 관리
/{bowling-id}/admin/accounts     → 어드민 > 관리자 계정 관리
```

## 9. 기술 요구사항

### 9.1 프론트엔드

- 칠텐 PC 기술 스택 연장: React 18 + TypeScript + Tailwind CSS
- 템플릿별 CSS 변수 시스템으로 테마 전환
- 이미지 최적화: WebP, lazy loading, responsive images
- SEO: SSR 또는 Pre-rendering (볼링장 검색 노출)

### 9.2 백엔드 API

- 칠텐 메인 API 서버와 통합 또는 마이크로서비스
- 볼링장별 데이터 격리 (multi-tenant)
- 이미지 업로드: AWS S3
- 알림 발송: FCM + 카카오 비즈메시지 API + SMS API
